## 外卖订餐

### 技术栈

- create-react-app 脚手架创建项目

- react + redux + react-router v4 + axios + less

- 采用`vw`移动端页面适配

## 项目运行

```sh
# 克隆项目
git clone git@github.com:linhuibin98/order-client.git

cd order-client

# 安装依赖
yarn install 或 npm install

# 运行
yarn start 或 npm start
```
